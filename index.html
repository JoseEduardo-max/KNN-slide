K-Nearest Neighbors (KNN)

Algoritmo baseado em proximidade

Hoje vou apresentar o KNN, um dos algoritmos mais simples e intuitivos do Machine Learning.

O que é KNN

  • Algoritmo de classificação (e regressão)
  • Baseado em similaridade
  • Usa os vizinhos mais próximos

O KNN classifica um novo dado com base nos dados mais próximos dele.

Intuição

  • "Diga-me com quem andas..."
  • Classificação por maioria

Se a maioria dos vizinhos for de uma classe, o novo ponto provavelmente será dessa mesma classe.

Como funciona

  1. Escolher K
  2. Calcular distância
  3. Selecionar vizinhos
  4. Votação

O algoritmo não aprende um modelo explícito, ele usa os dados diretamente.

Distância

d = √((x₁ - x₂)² + (y₁ - y₂)²)

Distância Euclidiana (mais comum)

A distância define quem é “vizinho” no problema.

O papel do K

  • K pequeno → sensível a ruído
  • K grande → mais estável

Escolher o K certo é essencial para o desempenho do modelo.

Vantagens

  • Simples
  • Fácil de entender
  • Sem treinamento complexo

Desvantagens

  • Lento com muitos dados
  • Sensível à escala
  • Escolha do K impacta muito

Quando usar

  • Problemas simples
  • Dados pequenos
  • Quando interpretação é importante

Conclusão

  • Baseado em proximidade
  • Fácil implementação
  • Muito usado como baseline

Apesar de simples, o KNN é muito útil e serve como base para entender outros algoritmos.
